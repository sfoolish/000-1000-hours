### [MacPorts](http://www.macports.org/)

#### 更新Port的index

    $ sudo port selfupdate 

#### 搜索索引中的软件

    $ port search <name>

#### 查看软件包的内容和说明

    $ sudo port info <name>

#### 查看软件包的依赖关系

    $ sudo port deps <name>

#### 安装新软件

    $ sudo port install <name>

#### 卸载软件

    $ sudo port uninstall <name>
