## Homebrew 安装

	$ ruby -e "$(curl -fsSL https://raw.github.com/Homebrew/homebrew/go/install)"

## Homebrew 基本功能

	# 软件安装
	$ brew install wget
	# 软件卸载
	$ brew uninstall wget
	# 软件查询
	$ brew search /wge*/
	brew list           列出已安装的软件

## Homebrew 其他命令

	$ brew update     ; 更新brew
	$ brew home       ; 用浏览器打开brew的官方网站
	$ brew info       ; 显示软件信息
	$ brew deps       ; 显示包依赖
