## Chrome 使用

### 快捷键

    * cmd + shift + b                   显示书签栏
    * cmd + d                           保存书签
    * ctl + shift + g (change colors)   背景色设置
    * cmd + shift + f                   全屏显示
    * cmd + alt + i                     开发者工具

### 常用插件

    * change colors
        颜色配置：
            Background color:       516154 / night    6D8271 / day
            Text color:             1DF00E
            Links color:            CFDB2E
            Visited links color:    FF7B17
